# Solar Constant Calculator

Проект на Qt 5 и CMake, вычисляющий солнечную постоянную (плотность потока излучения) у планеты. Вводятся радиус звезды в солнечных радиусах, температура поверхности в кельвинах и расстояние до планеты в астрономических единицах. В ответ выводится значение в Вт/м² через графический интерфейс Qt Widgets.

## Зависимости

На Ubuntu установите базовые пакеты Qt 5 и инструменты сборки, например:

```bash
sudo apt update
sudo apt install qtbase5-dev qtbase5-dev-tools qttools5-dev qttools5-dev-tools
```

## Сборка

```bash
cmake -S . -B build
cmake --build build
```

## Запуск

```bash
./build/solar_constant
```

Откроется окно, где можно задать радиус звезды (в солнечных радиусах), температуру поверхности (в Кельвинах) и расстояние до планеты (в астрономических единицах). Все значения должны быть положительными числами.

Добавьте параметр `--precision` (или кратко `-p`), чтобы указать количество
значащих цифр в выводе (от 1 до 15). По умолчанию используется 6 значащих цифр.
