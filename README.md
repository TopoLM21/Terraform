# Solar Constant Calculator

Проект на Qt 6 и CMake, вычисляющий солнечную постоянную (плотность потока излучения) у планеты. Вводятся радиус звезды в солнечных радиусах, температура поверхности в кельвинах и расстояние до планеты в астрономических единицах. В ответ выводится значение в Вт/м².

## Зависимости

На Ubuntu установите базовые пакеты Qt 6 и инструменты сборки, например:

```bash
sudo apt update
sudo apt install qt6-base-dev qt6-base-dev-tools qt6-tools-dev qt6-tools-dev-tools qt6-declarative-dev
```

## Сборка

```bash
cmake -S . -B build
cmake --build build
```

## Запуск

```bash
./build/solar_constant
```

Затем следуйте подсказкам в терминале (все вводимые значения должны быть положительными числами).

Альтернативно можно работать в неинтерактивном режиме, передав все параметры в командной строке:

```bash
./build/solar_constant --radius 1.0 --temperature 5772 --distance 1.0
```

Добавьте параметр `--precision` (или кратко `-p`), чтобы указать количество
значащих цифр в выводе (от 1 до 15). По умолчанию используется 6 значащих цифр.
